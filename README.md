# Corporate-Chat-AI-Spring

AI-powered corporate assistant built with **Spring Boot**, **n8n workflows**, and **Ollama agents**, designed to provide reliable company information through an interactive chat interface. Includes guardrails to prevent unsafe prompts and maintains structured, knowledge-grounded responses.

## Features
- **Modern chat UI** with animated thinking indicators and light/dark theme toggle
- **Spring Boot backend** serving frontend and handling webhook communication
- **n8n workflow integration** for orchestrating queries to Ollama agents
- **Guardrails** for input safety and prompt validation
- **Responsive design** with color palettes for professional branding

## Tech Stack
- Java / Spring Boot
- Frontend: HTML, CSS, Vanilla JS
- AI: Ollama agents
- Workflow & Webhooks: n8n
- Input Safety: Guardrails

## Usage
1. Run n8n workflow with your company webhook.
2. Start Spring Boot application.
3. Access frontend UI, ask company-related questions, and get dynamic AI responses.
